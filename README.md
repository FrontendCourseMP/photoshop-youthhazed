# Photoshop Lab: GrayBit-7

React-приложение для первой лабораторной по цифровому представлению изображения.

## Возможности

**Лабораторная 1 — представление изображения**
- загрузка PNG, JPG/JPEG и GrayBit-7 (`.gb7`);
- отображение изображения на HTML5 Canvas;
- скачивание изображения в PNG, JPG и GB7;
- ручная реализация кодера и декодера GrayBit-7;
- строка состояния с шириной, высотой и глубиной цвета.

**Лабораторная 2 — каналы, коррекция, масштаб, фильтры**
- панель каналов с миниатюрами (Gray/RGB/Alpha) и переключением видимости;
- пипетка: координаты пикселя, RGB (0–255) и CIELAB;
- инструмент «Уровни»: гистограмма (линейная/лог, по композиту и каналам),
  три маркера входных уровней с гамма-коррекцией, предпросмотр, Master/каналы;
- масштабирование собственной интерполяцией (ближайший сосед и билинейная),
  range-слайдер 12–300% и автоподгонка под холст;
- инструмент «Размер изображения»: проценты/пиксели, связь пропорций,
  выбор алгоритма интерполяции, валидация;
- фильтрация свёрткой 3×3 (резкость, Гаусс, размытие, Прюитт + своё ядро),
  выбор каналов, стратегии обработки краёв и расчёт в Web Worker.

