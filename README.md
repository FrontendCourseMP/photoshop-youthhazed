# Photoshop Lab: GrayBit-7

Статическое веб-приложение для первой лабораторной по цифровому представлению изображения.

## Возможности

- загрузка PNG, JPG/JPEG и GrayBit-7 (`.gb7`);
- отображение изображения на HTML5 Canvas;
- скачивание текущего холста в PNG, JPG и GB7;
- ручная реализация кодера и декодера GrayBit-7;
- строка состояния с шириной, высотой и глубиной цвета.

## Запуск

Откройте `index.html` в браузере или запустите любой статический сервер в корне проекта.
